# Module-5-python
This project is for module 5
This project demonstrates file handling and exception handling in Python.

## Task 1: Read a File and Handle Errors
- Opens and reads a text file named sample.txt
- Prints content line by line
- Handles FileNotFoundError gracefully

Run:
python task1_read_file.py

## Task 2: Write and Append Data to a File
- Takes user input
- Writes data to output.txt
- Appends additional data
- Displays final file content

Run:
python task2_write_append.py
